# InventoryHelper

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) adding inventory search and management features.
Based upon the [InventorySearch](https://github.com/ResoniteMods/InventorySearch) mod.

## Features
- Item and folder search in your inventory.
- Mod settings to search your entire inventory, or scope to the focused directory
- Copy, paste, and cut/move options for items and folders

## Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
2. Place [InventoryHelper.dll](https://github.com/noblereign/InventoryHelper/releases/latest/download/InventoryHelper.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create this folder for you.
3. Start the game. If you want to verify that the mod is working you can check your Resonite logs.
